## 什么是 ZenBox？

ZenBox 是一个现代化的 React 状态管理库，它将 Zustand 的简洁性与 Vue 的响应式模式完美结合。为熟悉这两个框架的开发者提供了直观自然的 API 体验。

**完美适合热爱 Zustand 和 Vue 的开发者！**

### 🐻 Zustand 的简洁性

```ts
const counter = createStore({ count: 0 });

counter.setState((state) => ({ count: state.count + 1 }));
```

### 💚 Vue 风格的响应式

```ts
const doubled = useComputed(() => counter.value.count * 2);

useWatch(
  () => counter.value.count,
  (current, prev) => console.log("变化了！", current, prev)
);
```

## 核心特性

- 🚀 **零学习成本** - 直观的 API，立即上手
- ⚡ **高性能** - 细粒度响应式更新，自动优化
- 💪 **完整 TypeScript 支持** - 开箱即用的完整类型推断
- 🎯 **灵活架构** - 支持全局和组件级作用域状态
- 📦 **轻量级** - 压缩后小于 3KB，[100 行核心代码](../blog/react-state-management-in-100-lines)
