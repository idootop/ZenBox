The ZenBox v1.0.0 is a major release that introduces a new way to manage state in React.

## Quick Start

```npm
npm install zenbox
```

## Example

```tsx
import { createStore, useStoreValue } from "zenbox";

const counterStore = createStore({ count: 0 });

const Counter = () => {
  const count = useStoreValue(counterStore);
  return <div>{count}</div>;
};
```
